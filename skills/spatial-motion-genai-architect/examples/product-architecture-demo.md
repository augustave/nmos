# Product Architecture Demo

## Scenario

A Series B developer tools company needs to explain a complex multi-tenant
orchestration layer to enterprise prospects. Static diagrams are failing to
convey the real-time behavior of their system.

## Input

- Technical architecture doc (service mesh, event bus, tenant isolation)
- Spatial interaction spec: gaze-driven navigation, hand-pinch to inspect nodes
- Brand constraints: dark UI, monospace type, minimal chrome
- Demo context: 3-minute walkthrough for CTO-level audience at a partner event

## Output

- 60-sec concept reel showing spatial data flow
- TouchDesigner node graph for real-time orchestration viz
- XR sandbox: walk through the architecture, pinch nodes to see latency/load
- Workflow doc for reproducing the prototype in Unity

## Review Notes

Keep the spatial metaphor honest — don't let the 3D environment imply more
system simplicity than exists. The goal is legibility, not spectacle.
